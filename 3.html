시장 세분화

STP분석

S(Segmentation, 시장세분화)
: 제공되는 제품과 서비스의 시장을 정의하고 소비자층을 세분화
: 지리적, 인구통계학적, 심리분석적, 행동적 특성 -> 시장 세분화
+ 고객의 나이, 성별, 라이프스타일, 가격민감도
> 인구 통계적
: 연령, 성별, 직업, 수입
ex) 40대 여성, 예비 부부, 수험생

> 지리적 구분
: 도시규모, 인구밀도, 기후
ex) 대치동(학군지), 강남(사무, 기업)

> 심리적 구분
: 성격, 생활 양식, 종교관
ex) 필리핀 or 캐나다(날씨) / 중동 or 유럽(종교)

> 행동적 구분
: 사용량, 빈도

= 제품을 구매할 고객을 특성별로 분류
--------------------------------------------------------------------------------------------------------

T(Targeting, 표적시장)
: 시장 세분화 -> 소비자 층중에서 자사의 제품/서비스를 선택할 가능성이 높은 핵심 소비자층 선별
: 성별 or 연령층으로 구분 -> 어떤 연령층에 제품이 구매될지 타깃 설정
= 기업이 집중해야할 주 고객 결정
*시장매력도(시장요인, 경쟁요인, 자사적합성 고려) -> 표적 시장 선정

>시장요인
1. 시장규모 : 현재 시장의 크기 파악
2. 시장성장성 : 미래 시장의 크기 변화 판단(커질지 or 작아질지)
3. 수익성 : 투자대비 수익이 높은지 낮은지 판단

> 경쟁요인
1. 현재의 경쟁현황 : 현재 시장이 경쟁이 치열한지(레드오션 or 블루오션)
2. 미래의 경쟁현황 : 앞으로 경쟁이 치열할지

>자사적합성
1. 기업목표 : 기업의 목표와 부합되는 시장인지
2. 자원 : 현재와 미래에 기업이 가진 자원이 충분한지
---------------------------------------------------------------------------------------------------------

P(Positioning, 포지셔닝)
: 선별된 목표 고객이 제품에 대해 어떻게 생각하는지, 경쟁품과 비교해서 어떤 포지션을 잡을 것인지 결정
: 시장에서의 차별점

ex) 제품 속성 우위 - 다른 제품에 없는 특별한 성분 추가 or 함량 증가 / 가격 경쟁력 높이기(가격 감소)
사용 상황 우위 - 갈증(숙취)해소 음료
이미지 우위 - 착한기업, 착한 제품(친환경)
제품 사용자 우위 - 아이들이 먹는 치즈, 청소년이 쓰는 립스틱 등등

*(매운라면? -> 신라면) 이런 방식으로 고유의 색을 만드는 방법
+ 침대는 과학이다, 커피는 T.O.P
